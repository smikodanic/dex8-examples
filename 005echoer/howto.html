005echoer

The example shows how to use Echoer methods log(), objekt(), error(), warn(), .... Every method should print message in different color.
Also it tests the behaviour of echoer.log() applied directly in the for loop, if it will work well in the web panel i.e. when echoer messages are sent via websocket.

Echoer methods

  • echoer.log('one', 2, 'three') - print string message(s)
  • echoer.objekt({x: 13, y: 'power'}) - print object
  • echoer.error(new Error('My error!')) - print error
  • echoer.warn('Some warning') - print warning
  • echoer.image(img_b64) - send image in base64 format
  • echoer.listen() - listen for an input from the Web Panel
