003input_object

Example with object in the input.json echoed by await echoer.objekt() method.
The input also contains the password field which will be encoded when uploaded to dex8 Platform.
For example: 

input.json
==============================
{
  "myPassword": "341Wqe",
  "myObject": {
    "a": 1,
    "x": "Some string"
  }
}
