003input_object

Example with object in the input.json echoed by echo.objekt() method.

The example shows that password field will be encoded and it will not affect on object field if both fields are in input file.
For example: 

input.json
==============================
{
  "myPassword": "341Wqe",
  "myObject": {
    "a": 1,
    "x": "Some string"
  }
}

DEX8-SDK CLI

$ dex8 start -i input.json
