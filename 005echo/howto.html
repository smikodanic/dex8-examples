005echo

The example shows how to use Echo helper methods log(), objekt() and error().

Echo methods

  • echo.log('one', 2, 'three') - print string message(s)
  • echo.objekt({x: 13, y: 'power'}) - print object
  • echo.error(new Error('My error!')) - print error
  • echo.image(img_b64) - send image in base64 format
  • echo.listen() - listen for an input from the Web Panel
