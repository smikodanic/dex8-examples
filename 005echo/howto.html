005echo

The example shows how to use Echoer methods log(), objekt(), error(), warn(), .... Every method should print message in different color.
Also it tests the behaviour of echo.log() applied directly in the for loop, if it will work well in the web panel i.e. when echo messages are sent via websocket.

Echoer methods

  • echo.log('one', 2, 'three') - print string message(s)
  • echo.objekt({x: 13, y: 'power'}) - print object
  • echo.error(new Error('My error!')) - print error
  • echo.warn('Some warning') - print warning
  • echo.image(img_b64) - send image in base64 format
  • echo.listen() - listen for an input from the Web Panel
