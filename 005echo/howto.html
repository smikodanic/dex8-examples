005echo

The example shows how to use Echo helper methods log(), objekt(), error(), warn(), ....
Also it tests the behaviour of echo.log() applied directly in the for loop. Especially if it will work well via web panle i.e. via regoch-websocket.

Echo methods

  • await echo.log('one', 2, 'three') - print string message(s)
  • await echo.objekt({x: 13, y: 'power'}) - print object
  • await echo.error(new Error('My error!')) - print error
  • await echo.warn('Some warning') - print warning
  • await echo.image(img_b64) - send image in base64 format
  • await echo.listen() - listen for an input from the Web Panel
