005echo

The example shows how to use Echo helper.

Echo methods

  • echo.log('one', 2, 'three') - print string message(s)
  • echo.objekt({x: 13, y: 'power'}) - print object
  • echo.error(new Error('My error!')) - print error
  • echo.send({msg, obj, err}) - send message, object and error in same method
