005echo

This example shows how to use Echo methods log(), objekt(), error(), warn(), ....
Every method should print message in different color.
Additionally, it evaluates the functionality of echo.log() when applied directly within the for loop to ensure regular operation within the web panel, particularly when echo messages are transmitted via websocket.

@mikosoft/echo methods

  • echo.log('one', 2, 'three') - print string message(s)
  • echo.objekt({x: 13, y: 'power'}) - print object
  • echo.error(new Error('My error!')) - print error
  • echo.warn('Some warning') - print warning
  • echo.image(img_b64) - send image in base64 format
  • echo.listen() - listen for an input from the Web Panel
