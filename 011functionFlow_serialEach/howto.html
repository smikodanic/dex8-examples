FunctionFlowX serialEach(funcs, arr) method test. Functions funcs will be executed serially and it will be repeated or every array arr element.

How to access FunctionFlow object in the function?

A) First way, like it's done in f1.js, the ff is accessed with const ff = global.functionFlow;. There's no need to inject ff in the main.js.

B) Inject ff with ff.libInject(lib); in the main.js and access it with const ff = lib.ff; in the f2.js.
