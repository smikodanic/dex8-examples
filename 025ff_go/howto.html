025ff_go

FunctionFlow go(ind) method.
Go to function defined in serial(funcs) method. When that function is executed continue with next function in funcs array.

In this example there are 7 functions f0 - f6.
Function f2 has go(1) what is returning flow to f1 until x is smaller then 13.
Function f4 has go(6) what jump over f5 and execute f6.
