050infinite_run

The example repeats 50000 times function func1. The task is suitable to test "start", "stop" and "pause" actions.




Tests:

  1. If start button is pressed twice it should show error 'Task is already running'.
  2. First press start then stop. It should stop further execution of this task.
  3. First press start then pause. It should pause task execution. Resume by pressing start button again.
