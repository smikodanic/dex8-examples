026ff_jump

FunctionFlow jump() method.
Jump to iteration number defined in repeat(n) method. When that iteration is executed continue with next iteration in repeat(n) method.

In this example there are 3 functions f1 - f3.
Serial method is should be repeated 10 times, but function f2 has jump(10) so on 2nd repeat iteration will jump to last iteration.
