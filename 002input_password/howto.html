002input_password

When input file contains "password" keyword (case insensitive) in its name it is password field. For example: password, PASSWORD, PASSword, password3, my_Password8, ...
All fields which contain "password" string are encoded in database due to security.
In this example input.js contains 3 password fields: password, password_1 and MyPAssword_1 which are encoded in the database but decoded when printed in the console.

DEX8-SDK:

$ dex8 start -i input.json -s
